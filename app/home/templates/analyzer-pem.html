ASN.1(Abstract Syntax Notation) 데이터

    ASN.1은 전화망과 컴퓨터 네트워킹에서 데이터를 표현, 인코딩, 전송, 디코딩하기 위한 규칙과 구조체를 기술하는 표준과 표기법이다. 형식적인 규칙은 객체의 표현을 머신에 특화된 인코딩 기술에 독립적이 되도록 한다. 형식적인 표기법은 데이터 표현을 한 특정 인스턴스가 스펙을 바로 준수했는지 여부를 검증하는 업무를 자동화할 수 있다. 즉, 소프트웨어 도구가 검증에 사용될 수 있다는 것이다.

    ASN.1은 ISO와 IEC, 그리고 ITU-T가 합동으로 표준화한 것으로 CCITT X.409:1984의 부분으로 1984년에 본래 정의된 것이다. ASN.1은 널리 사용되면서 1988년에 X.208로 다시 표준화되었다. 실질적으로 수정된 1995 버전은 X.680 시리즈에 의해 포괄되었다. X.680 시리즈의 최신 수정은 5.0판이며 2015년에 출판되었다.

    ASN.1은 정보에 대한 추상 구문을 정의하지만 정보가 인코딩되는 방식을 제한하지는 않는다. 다양한 ASN.1 인코딩 규칙이 추상 구문이 ASN.1으로 기술된 데이터 값에 대한 전송 구문(구체적인 표현)을 제공한다.

    표준 ASN.1 인코딩 규칙은 아래와 같다.

  • Basic Encoding Rules (BER)
  • Canonical Encoding Rules (CER)
  • Distinguished Encoding Rules (DER)
  • XML Encoding Rules (XER)
  • Canonical XML Encoding Rules (CXER)
  • Extended XML Encoding Rules (E-XER)
  • Packed Encoding Rules (PER, 정렬된 것: UPER, 표준: CPER)
  • Generic String Encoding Rules (GSER)
